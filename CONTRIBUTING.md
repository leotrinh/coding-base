# Contributing

Thanks for contributing.

This repository is intended to be a public engineering standard pack.

## Rules for contributions

1. Distinguish **official conventions** from **team defaults**.
2. Prefer primary sources:
   - official docs
   - language style guides
   - framework reference docs
   - vendor documentation
3. Optimize for code review:
   - clear
   - enforceable
   - easy to scan
4. Keep examples realistic.
5. Do not over-engineer the standards themselves.

## Pull request checklist

- wording does not overclaim
- official vs opinionated recommendation is clear
- examples use consistent naming
- no mixed naming style inside one example
- content is readable by an external reviewer
